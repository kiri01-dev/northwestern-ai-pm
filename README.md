# Northwestern AI PM Talk Deck

Live site: https://<your-github-username>.github.io/northwestern-ai-pm/

This repo hosts a single self-contained HTML slide deck (`index.html`) via GitHub Pages.

## Updating the deck

1. Export/copy the new version of the deck over `index.html` in this folder.
2. From this folder, run:
   ```
   git add -A
   git commit -m "Update deck"
   git push
   ```
3. GitHub Pages rebuilds automatically, usually live within ~1 minute.

## One-time setup (see setup instructions from Claude for full detail)

- Create the GitHub repo and push this folder.
- In the repo: Settings -> Pages -> Source: Deploy from branch -> main -> /(root) -> Save.
